# Markdown 入门指南

## 什么是 Markdown？

**Markdown** 是一种轻量级标记语言，创始人为约翰·格鲁伯（John Gruber）。它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的 HTML 文档。

## 为什么使用 Markdown？

1. **易于学习** - 语法简单，几分钟即可上手
2. **广泛支持** - GitHub、GitLab、Notion 等都支持
3. **格式统一** - 不同平台渲染效果一致
4. **纯文本** - 可以用任何文本编辑器打开

## 基础语法

### 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 列表

无序列表：

```markdown
- 项目 1
- 项目 2
- 项目 3
```

有序列表：

```markdown
1. 第一步
2. 第二步
3. 第三步
```

### 代码

行内代码：`` `code` ``

代码块：

````markdown
```swift
print("Hello")
```
````

## 进阶技巧

> 提示：使用表格可以更好地组织数据。

| 场景 | 推荐用法 |
|------|----------|
| 笔记 | 标题 + 列表 |
| 文档 | 完整结构 |
| 代码 | 代码块必不可少 |

## 总结

Markdown 让写作变得简单而优雅。开始使用吧！
